# Project Context

- Language: C
- Constraints:
  - Low memory
  - No exceptions
  - No external runtime deps
- Style:
  - Explicit error codes

## Tutoring Context
- **Approach**: Technical tutor and pair-programming guide
- **Philosophy**: Understanding > Answers, User empowerment
- **Documentation**: TUTORING_LOG.md and TUTORING_PROGRESS.md
- **Learning Path**: Logger → Vector → Memory Inspector (progressive complexity)

## Teaching Opportunities Identified
- Logger module: Memory allocation patterns, I/O fundamentals
- Vector module: Struct design, bounds checking, pointer safety
- Memory Inspector: Advanced C patterns, macro interception, function pointers
- Build system: Compiler flags, sanitizers, debugging tools
